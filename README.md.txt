# Football Portal Web Application

## Project Description
A comprehensive football portal web application that provides information about matches, teams, news, and more.

## Project Structure
```
football-portal/
├── frontend/           # Frontend application files
│   ├── css/           # Stylesheets
│   ├── js/            # JavaScript files
│   ├── views/         # HTML view files
│   ├── assets/        # Static assets (images, fonts)
│   └── index.html     # Main SPA entry point
│
└── backend/           # Backend application files
    ├── routes/        # API routes
    ├── services/      # Business logic
    └── dao/           # Database access objects
```
