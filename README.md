# 90minut Football Portal

**Milestone 5: Final Deployment, API Documentation, and CI/CD**

Frontend je implementiran po MVC principu (samo `Services` folder sa JS fajlovima), uz client-side i server-side validacije i sigurnosna poboljšanja.  
Aplikacija je deployana na DigitalOcean, baza je postavljena i povezana, ali login i AJAX pozivi trenutno ne rade zbog konfiguracije servera ili hostinga.
